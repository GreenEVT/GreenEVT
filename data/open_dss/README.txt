This folder contains the network data for each substation (and subtransmission level) in the urban-suburban portion of the Greensboro NC data set. 
It also contains the OpenDSS bus coordenates (Buscoords.dss) and the OpenDSS Master file for all the network in the folder (Master.dss).
It also contains some analysis that came together with the data (analysis folder).
It also contains a spreadsheet with data on each bus on the network (bus_info.csv) and text file with the longitude and latitude data for each bus (Long_lat_buscoords.txt).

Each feeder contains multiple Load.dss files that represent the loads under different EV penetration scenarios.
